## English
The data set is located entirely in the file Rest_Mex_Sentiment_Analysis_Test_2023.xlsx
The data set consists of 107,863 rows (30% of the original data set), one for each opinion. Each row contains 3 columns:

1. Id: The identifier of each instance, this data is important since it is the one with which each instance will be identified for the results.
2. Title: The title that the tourist himself gave to his opinion. Data type: Text
3. Opinion: The opinion issued by the tourist. Data type: Text

## Spanish (Español)
El conjunto de datos se encuentra completamente en el archivo Rest_Mex_Sentiment_Analysis_Test_2023.xlsx
El conjunto de datos consta de 107,863 renglones (el 30 % del conjunto de datos original), uno por cada opinión. Cada renglón contiene 3 columnas:

1. Id: El identificador de cada instancia, este dato es importante ya que es con el que se identificará cada instancia para los resultados.
2. Title: El título que el propio turista le otorgó a su opinión. Tipo de dato: Texto 
3. Opinion: La opinión emitida por el turista. Tipo de dato: Texto
